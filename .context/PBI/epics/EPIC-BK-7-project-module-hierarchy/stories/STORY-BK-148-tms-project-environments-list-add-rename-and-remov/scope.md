# BK-148 — Scope

> Jira field: `customfield_10119` · [View in Jira](https://jira.upexgalaxy.com/browse/BK-148)

- List every environment configured for the current project, in a stable order.
- Add a new environment to a project by giving it a name.
- Rename an existing environment of the project.
- Remove an environment from the project when it is safe to do so.
- Enforce a unique environment name within the same project (case-insensitive, trimmed).
- Trim surrounding whitespace from the name and enforce a sensible length on add and rename.
- Show a clear confirmation before an environment is removed.
- Surface a clear, in-context message when an add, rename, or remove is rejected.
