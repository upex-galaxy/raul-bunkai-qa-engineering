# BK-147 — Scope

> Jira field: `customfield_10119` · [View in Jira](https://jira.upexgalaxy.com/browse/BK-147)

- After sign-in, the application shell — left navigation, workspace switcher, global search entry, and the account block — stays visible on every screen, so navigation context is never lost.
- Inside a project, the explorer tree stays visible on every project view: the workbench index, an open ATC, and an open Test — not only the workbench index.
- Opening an ATC or a Test opens it as a tab in the workbench; several tabs can be open at once, switchable and closable, with the explorer beside them.
- The project toolbar (New ATC, New Test, view switch, search/filter) is reachable from any open tab.
- A direct link to an ATC or a Test opens it as a tab with the explorer visible.
- The item currently open is highlighted in the explorer, for both ATCs and Tests.
