## User story

***As a*** Senior QA Engineer
***I want to*** write User Story and Acceptance Criterion descriptions in Markdown with a small editor and live preview, rendered safely
***So that*** the content reads well to people and stays clean and AI-readable for downstream agents, with no risk of unsafe content running in the app.
